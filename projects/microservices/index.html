Deploy applications as Microservices to AKS | Terraform | Azure DevOps

October 25, 2023

  • DevOps
  • Kubernetes

Working on a project where multiple applications are getting deployed as containers. AKS is used as the container orchestration platform, Azure Container Registry (ACR) is used to store the container images. Container images are built using Docker. Build pipeline is created in Azure DevOps and jobs are run using Self-Hosted Linux Agent server.

Below Infrastructure components were created using Terraform

  • Azure Linux VM (RHEL)
  • Azure Data Factory
  • Azure Windows VM(Self Hosted Integration Runtime for ADF)
  • Azure Cosmos DB (PGSQL)
  • Azure Storage Account
  • Azure Container Registry
  • Private Endpoints for ACR,Storage Account, PGSQL, ADF
  • Azure Key Vault
  • Private Azure Kubernetes Cluster with Application Gateway Ingress Controller and Key Vault Integration

Linux VM was configured as Self-Hosted Agent server to run Azure DevOps pipeline for building Docker images. Argo CD was used to deploy application and other components to AKS using GitOps principles.
